# PRP Security  
Pseudo-random permutation security.  
[Block cipher](../Block_cipher.md) is PRP secure when the [Advantage](./Advantage.md) of distinguishing it from a [Random Permutation](./Random_Permutation.md) is $\leq 2^{-n}N$.
